# Mini Security Toolkit

A modular Python-based security toolkit built for learning defensive cybersecurity concepts and Python development.

## Features

- Port Scanner
- System Resource Monitor
- File Integrity Checker
- Linux Log Analyzer

## Technologies

- Python
- Tkinter
- Socket Programming
- Hashing (SHA-256)
- psutil
